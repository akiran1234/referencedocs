# Elastic Load Balancer (ELB)
# A load balancer which acts as an  Internet Facing and servers  client requests. 
# Classic Load Balancer Vs Application Load Balancer
# In classic load balancer the registered targets will have same content (static) across all instances and load/requests are served in round roubin fashion.
# In Application Load Balancer the request can be served to a particular target host i.e path based routing. In essence it is content based request.
